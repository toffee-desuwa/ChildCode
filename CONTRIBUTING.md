# Contributing to ChildCode

Thanks for your interest in contributing to **ChildCode — Scratch for the AI Era**! We welcome contributions of all kinds.

## Getting Started

```bash
# Fork and clone the repo, then:
npm install
npm run dev
```

The app runs at `http://localhost:5173` by default.

## Submitting Changes

1. Fork the repository
2. Create a feature branch (`git checkout -b my-feature`)
3. Make your changes
4. Run `npm run lint` to check code style
5. Commit and push to your fork
6. Open a Pull Request against `main`

Please keep PRs focused on a single change and include a clear description of what you did and why.

## Code Style

We use ESLint for linting. Before submitting, run:

```bash
npm run lint
```

## Reporting Issues

Found a bug or have an idea? Open an issue using our templates — we'd love to hear from you.

## Questions?

Feel free to open a discussion or issue. We're happy to help!
